JavaScript ES6 Promise Demo
============================

刚明白ES6和ES2015指的是同一个规范，`Promise`是在这一个规范中增加的。

```
npm install
npm run demo
```

It will print:

```
Promise { <pending> }
promise is created.
promise result: 3
```

需要注意的是：`promise is created.`这一行输出完3秒钟以后，才输出`promise result: 3`这一行。
